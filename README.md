# iamgodiam-mesh

The **Sovereign Web Mesh** — a 10,038-page static knowledge architecture published at **[iamgodiam.net](https://iamgodiam.net)**. Agentic-generated, organized by a 42-agent board, anchored on the Base blockchain.

## What it is
20 knowledge clusters — Urantia Book study, EDEN philosophy, Black Dragon Initiative, AI architecture, sovereign game lore, social justice & AI, grant templates, NFT universe, Hallways docs, and more.

## Stack
Static HTML/CSS/JS (no build step) · canvas starfield · GitHub Pages + CNAME → iamgodiam.net · generated by the `atlas-10k-engine` (metadata + Merkle root in `manifest.json`).

## Deploy
The committed HTML **is** the site — "the site is the artifact." Do **not** strip `pages/`. GitHub Pages serves from the default branch.

## Status
Live (10,038 pages, 0 generation errors). The public knowledge face of the IAMGODIAM org; distinct from `e5-website` (the institutional nonprofit site).
